DESIGN AND ANALYSIS OF ALGORITHMS

Course Details

Linear Search Algorithms

Sequential searching technique that checks each element one by one.

Sorting & Searching Algorithms

Techniques used to organize and efficiently locate data.

Divide and Conquer

Splits the problem into smaller subproblems, solves them independently, then combines results.

Graph Algorithms

Algorithms for traversing, searching, and processing graph structures.

Greedy Algorithms

Choosing the best immediate option in each step to reach a global solution.

Dynamic Programming

Solving problems using memoization and overlapping subproblems.
